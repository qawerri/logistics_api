# Logistics REST API

RESTful API для автоматизации работы логистической компании. Система управляет парком транспорта, водителями, заказами на перевозку и предоставляет возможность отслеживания статусов доставки в реальном времени.

## Функционал
- Полный CRUD для сущностей: водители, транспорт, заказы, события отслеживания
- JWT-аутентификация (access/refresh токены)
- Гибкая фильтрация и поиск по параметрам (адрес, статус, вес, тип ТС и т.д.)
- Пагинация списков и троттлинг запросов (защита от перегрузки)
- Кэширование GET-запросов для ускорения отдачи данных
- Ролевая модель прав доступа (чтение открыто, запись/изменение только для авторизованных)
- Автоматическая интерактивная документация Swagger/OpenAPI
- Встроенная Django Admin-панель для ручного управления данными
